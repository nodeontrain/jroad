# Jroad
A middleware framework for node
[![NPM](https://nodei.co/npm/jroad.png)](https://nodei.co/npm/jroad/)

## Install Jroad
```shell
npm install jroad
```

## Update Jroad
```shell
npm update jroad
```

## Running Tests
Before you run tests, you may need to first install mocha module and jspec module.
```shell
sudo npm install -g mocha
npm install jspec
```

To run the tests:
```shell
npm test
```
